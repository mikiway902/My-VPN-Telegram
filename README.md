# My-Second-VPN

## Использумые библиотеки:
  Мой развивающийся проект, совмещающий между собой **Telegram Бота** написанного с использованием асинхронных библиотек **aiogram**, **asyncio**, базы данных **PostgreSQL** и сетевого инструмента **Xray-Core**, **dotenv** для хранения API ключей.

## Реализация проекта на сегодняшний день:
  На данный момент готова часть телеграмм бота с использованием запросов CREATE, SELECT, INSERT, UPDATE, WHERE, SET.
