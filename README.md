# My-VPN-Telegram

## Суть проекта:
  Основным фактором стоит удобство развертывания VPN-сервера с использованием Docker compose исключительно для рабочих задач. 
  К примеру, подключение к рабочему серверу находящемуся в одной стране без использования доменов и удобным добавлением новых пользователей с ведением базы данных пользователей.
  Контейнеризация позволяет создать микросервисную архитектуру приложения для повышения стабильности и устойчивости полученной системы, контроля и масштабирования.

## Использумые библиотеки:
  Мой развивающийся проект, совмещающий между собой **Telegram Бота** написанного с использованием асинхронных библиотек **aiogram**, **asyncio**, базы данных **PostgreSQL** и сетевого инструмента **Xray-Core**, **dotenv** для хранения API ключей.

## Реализация проекта на сегодняшний день:
  На данный момент готова часть телеграмм бота с использованием запросов CREATE, SELECT, INSERT, UPDATE, WHERE, SET.
  Использование Xray-Core будет реализовано в будущем.
